# PRY_TAREA_SITIO_WEB
Desarrollar una página o sitio web dinámico aplicando los principios de diseño web semántico, el uso de Bootstrap como framework de diseño, y la incorporación de contenidos multimedia, formularios y componentes interactivos. Además, se debe integrar la funcionalidad de consumo de datos mediante Fetch API.


##  Cómo ejecutar

No requiere instalación

1.	Clona el repositorio.
2.	Abre el archivo index.html con live server:
3.	Se abrirá en tu navegador por defecto.
